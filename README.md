
DMD GIF Converter 128x32 - moteur comparatif v2.0
Shan_ayA 2026

Application complète de conversion d'images en GIF optimisés pour écrans DMD 128x32
avec moteur comparatif, édition manuelle avancée et génération de texte animé.

Dépendances:
    pip install pillow numpy
