## AUTO Tab 🤖

### Overview  
The **AUTO** tab enables automatic conversion of images into GIFs optimized for DMD 128x32 displays, with global settings and AI-generated proposals.

### Features

- 📁 **Select Folder**: Choose a source folder containing images.  
- 🖼️ **Select Images**: Manually select images to convert.  
- ✔️ **Recursive**: Include subfolders during selection.  
- ⚙️ **Global Parameters**:
  - FPS (frames per second) ⏱️  
  - Minimum animation duration in seconds ⌛  
  - Scroll speed (pixels per frame) 🌀  
  - Contrast, Saturation, and GIF color count 🎨  

- 🖼️ **Image List** with options to select all, none, or invert selection.  
- 🔓 **Reauthorize Image** to reactivate processing of images excluded in MANUAL mode.  
- 💻 **Preview** of the original image and the optimized DMD 128x32 output.  
- 💡 **AI Proposals** displayed as thumbnails with locking option.  
- 🚀 **Buttons**:
  - Process entire folder or selected images  
  - Process only selected images

### Tips  
- Select your folder or images and click "Process All" to automatically generate optimized versions chosen by the comparison engine.  
- Lock an AI proposal to apply it to all images (excluding the comparison engine).  
- If you select an image and switch to the MANUAL tab, it will be displayed automatically and excluded from automated processing when generating a GIF.

## MANUAL Tab ✍️

### Overview  
The **MANUAL** tab offers advanced manual editing tools for images, with effect application and custom animation.

### Core Features

- 📂 **Load Image** for manual editing.  
- ✂️ **Crop 128×32**: Define a precise cropping area.  
- ↶ **Undo**: Allows reverting the last modification (history preserved).  
- 💾 **Export GIF**: Save your work.  
- 📚 **Multi-images**: Load up to 4 images for morphing animations.  
- 🎬 **Morphing**: Generate smooth transition animations between images.

### Real-time Effects 🖌️

- Adjust brightness, contrast, saturation, sharpness via sliders.  
- Popular filters: blur, gaussian blur, edges, emboss, detail+, invert, horizontal/vertical mirror, rotate 90°, grayscale, solarize, posterize, equalize, autocontrast.  
- Drawing tools: fill 🎨 and magic eraser 🧹 with color and tolerance selection.

### Animation & Parameters

- Animation types: scroll, fade, zoom, rotate, wave, bounce, flash, slide, spiral, shake, pulse, glitch, pixelate, blur transition, color shift.  
- Set FPS, speed, duration, loop type (normal, ping-pong, infinite), and repetitions.  
- Advanced controls: easing, start delay, reverse, opacity.

### Preview

- Live animation preview panel on the right.  
- Detailed status and image info.

## TEXTSCROLL Tab 📝

### Overview  
Create and animate scrolling text optimized for DMD 128x32.

### Features

- 📝 **Text box** for input (practically unlimited length).  
- 👩‍🎨 **Font customization**:
  - Family (complete system font list)  
  - Size  
  - Bold and Italic styles  
  - Text and background colors selectable via color pickers.  
- 🎨 **Text visual effects**:
  - Normal, 3D, fire, snow, ice, metal, neon, graffiti, pixel art, outline, shadow.  
- 🌈 **Color effects**:
  - Rainbow, matrix, fire, gradient, none.  
- 🔄 **Text animations**:
  - Horizontal/vertical scroll, wavy scroll, Star Wars style, bounce, typewriter, explode, matrix rain, spiral, shake, glitch, fade, static.  
- ⏱️ **Animation settings**:
  - FPS, speed, duration  
  - Auto-adjust option.

### Controls

- 🎬 **Generate Preview** of the text animation.  
- 💾 **Export GIF** to save the animated text.

## SETTINGS Tab ⚙️

### Global Application Settings

- 🌍 **Language**: French 🇫🇷, English 🇬🇧, Spanish 🇪🇸.  
- 🎨 **Appearance**: Dark theme (default) or light theme.  
- ⚙️ **Behavior**:
  - Option to add animation type to exported filename.  
- 🎨 **Default GIF quality**: Color counts [8,16,32,64,128,256].  
- ⚡ **Performance**: Enable/disable AI cache.  
- 🗑️ **Cache**: Button to clear cache.  
- 📜 **Logs**: Options to auto-save, export, or clear logs.
